APSRTC BUS TICKETING SYSTEM - PROJECT REPORT
===========================================

1. PROJECT OVERVIEW
------------------
The APSRTC Bus Ticketing System is a comprehensive web-based solution designed to modernize and streamline the bus ticketing operations of the Andhra Pradesh State Road Transport Corporation (APSRTC). The system provides a complete suite of features for bus management, ticket booking, and administrative functions.

2. TECHNICAL ARCHITECTURE
------------------------
2.1 Frontend Technologies:
    - Next.js 15
    - React 19
    - TailwindCSS for styling
    - Various UI components and libraries

2.2 Backend Technologies:
    - Next.js API Routes
    - Prisma ORM for database operations
    - PostgreSQL database
    - NextAuth.js for authentication

2.3 Key Dependencies:
    - @prisma/client: Database ORM
    - @auth/prisma-adapter: Authentication adapter
    - @tanstack/react-query: Data fetching and caching
    - chart.js & recharts: Data visualization
    - Various ML libraries for analytics
    - QR code generation capabilities

3. CORE FEATURES

3.3 Vendor POS System:

3.4 Admin Dashboard:
